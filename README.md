# Media Library Tool

Инструмент для автоматической организации фото и видео архивов.

## Возможности

*   **Исправление дат**: Автоматически извлекает дату съемки из имени файла (поддержка Android, iOS, WhatsApp, Signal) и прописывает её в EXIF.
*   **Сдвиг времени**: Коррекция дат для камер со сбитыми часами.
*   **Организация папок**: Переименование папок по дате события (например, `2023-09 Отпуск`).
*   **Интерполяция**: Восстановление дат для файлов без метаданных на основе соседних файлов.

## Установка

1.  Установите [Python 3.10+](https://www.python.org/).
2.  Скачайте [ExifTool](https://exiftool.org/) и распакуйте в папку `exiftool` рядом со скриптом (или укажите путь в настройках).
3.  Установите зависимости:
    ```bash
    pip install customtkinter
    ```

## Запуск

```bash
python MediaLibraryTool.pyw
```
